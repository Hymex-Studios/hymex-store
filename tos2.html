Terminos y Condiciones del Cliente

Para comprar un producto debes de hacerlo a través de nuestra tienda oficial ,puedes abrir un ticket y en este los encargados en responder los tickets te ayudaran dando información básica si es necesario Una vez escogido el producto que quiere adquirir, solo tendría que esperar a que se le notifique a nuestro equipo para comiencen con el pedido No hay un limite de tiempo de entrega, el producto será entregado cuando este sea acabado. Si el cliente exige un tiempo de entrega especifico se le cobraría un % mas, quiere decir que en cuanto menos tiempo exija se ira subiendo el precio del producto Si el producto ya entregado no llega a satisfacer al cliente (no esta hecho como pidió, etc...) puede dar una mala reseña o puede ser paciente y mejorárselo hasta que cliente quede satisfecho. Al comprar un producto/servicio deben de confirmarnos el uso que le van a dar, ya sea uso personal o uso público (revenderlo etc...) Una vez comprado el producto en nuestra tienda oficial pueden abrir un ticket si tienen algún descuento para devolverles el % del descuento. Una vez comprado algún producto de la WEB que sea personalizable deberían de abrir ticket para comentarnos como lo quieren, después de ello se le pedirá mas dinero dependiendo de las cosas añadidas al producto que viene por defecto.
